# 📱 Smart Cart Supermarket — Mobile Application

Application mobile développée dans le cadre du Projet de Fin d'Études (PFE) Smart Cart.

## 🚀 Technologies

- React Native
- Expo
- TypeScript
- React Navigation
- Context API / Store
- Axios

## ✨ Fonctionnalités

- Authentification
- Consultation des produits
- Gestion du panier intelligent
- Scan RFID/QR (selon le projet)
- Historique des achats
